# Petvago

https://petvago.site/

# Specification

1. Search
    * Filter by location, pet type, date
2. Booking
3. Chatroom
    * Redis queue
4. Map
5. Login 
    * JWT, facebook, wechat
    * bcrypt
    * session
6. Review

# Restful API

1. /auth
    * For authentication
1. /api/userprofile
    * For user retreiving user information
1. /api/hotel
    * For hotel information
1. /api/search
    * For search input and result, and filter   
1. /api/chatroom
    * For chatroom conversation
1. /api/booking
    * For booking
1. /api/review
    * For review
